# Target Utils

Utilidades comunes para desarrollos de Targets, incluyendo mapeo de códigos IATA, clasificación de vuelos (doméstico, interlínea, etc.) y lógica reutilizable.

## 📋 Descripción

Este repositorio centraliza funcionalidades reutilizables para operaciones relacionadas con vuelos, aeropuertos y códigos internacionales IATA. Proporciona herramientas para mapeo de códigos, clasificación de tipos de vuelo y gestión de datos relacionados.
